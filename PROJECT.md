# Referent

Веб-приложение для анализа англоязычных и русскоязычных статей с помощью AI.

1. Приложение получает на входе адрес англоязычной или русскоязычной статьи, выполняет парсинг и, по соответствующей кнопке, подключает AI и выполняет нужное действие.

2. Возможности:
   - Парсинг содержимого статьи по URL
   - Генерация краткого описания статьи
   - Генерация изображения под статью
   - Анализ на плагиат
   - Формирование тезисов
   - Создание поста для Telegram

3. Интерфейс:
   - Поле ввода URL англоязычной или русскоязычной статьи
   - 5 кнопок: «О чем статья», «Изображение», «Плагиат», «Тезисы», «Пост для Telegram»
   - Блок для отображения результата:
     - текстовый ответ (описание, тезисы, пост)
     - процент плагиата
     - сгенерированное изображение

4. Стек:
   - Next.js
   - React
   - TypeScript
   - Tailwind CSS
