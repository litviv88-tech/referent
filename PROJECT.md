# Referent

Веб-приложение для анализа англоязычных и русский  статей с помощью AI.

1. Приложение получает на входе адрес англоязычной статьи или русской статья, выполняет парсинг и, по соответствующей кнопке, подключает AI и выполняет нужное действие.

2. Возможности:
   - Парсинг содержимого статьи по URL
   - Генерация краткого описания статьи
   - Генерация изображения под статью
   - Анализ на плагиат
   - Формирование тезисов
   - Создание поста для Telegram

3. Интерфейс:
   - Поле ввода URL англоязычной статьи
   - 3 кнопки: «О чем статья», «Тезисы», «Пост для Telegram»
   - Блок для отображения результата, в котором происходит генерация нужного вида ответа
    - процент плагиата
     - картинка 

4. Стек:
   - Next.js
   - React
   - TypeScript
