# Referent

Веб-приложение для анализа англоязычных статей с помощью AI.

Приложение получает на входе адрес англоязычной статьи, выполняет парсинг и, по соответствующей кнопке, подключает AI и выполняет нужное действие.

## Возможности

- Парсинг содержимого статьи по URL
- Генерация краткого описания статьи
- Формирование тезисов
- Создание поста для Telegram

## Интерфейс

- Поле ввода URL англоязычной статьи
- 3 кнопки: **«О чем статья»**, **«Тезисы»**, **«Пост для Telegram»**
- Блок для отображения результата, в котором происходит генерация нужного вида ответа

## Стек

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)

## Запуск

```powershell
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```powershell
npm run build
npm start
```
