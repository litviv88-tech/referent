# Referent

Веб-приложение для анализа англоязычных и русскоязычных статей с помощью AI.

Приложение получает на входе адрес статьи, выполняет парсинг и, по соответствующей кнопке, подключает AI и выполняет нужное действие.

## Возможности

- Парсинг содержимого статьи по URL
- Генерация краткого описания статьи («О чем статья»)
- Формирование тезисов
- Создание поста для Telegram
- Перевод статьи на русский язык
- Генерация изображения под статью
- Анализ на плагиат

## Интерфейс

- Поле ввода URL англоязычной или русскоязычной статьи
- 6 кнопок: **«О чем статья»**, **«Тезисы»**, **«Пост для Telegram»**, **«Перевод»**, **«Изображение»**, **«Плагиат»**
- Блок для отображения результата:
  - текстовый ответ (описание, тезисы, пост)
  - переведённая статья (заголовок и текст)
  - процент плагиата
  - сгенерированное изображение

## Стек

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)

## Настройка

Скопируйте `.env.example` в `.env.local` и укажите ключ OpenRouter:

```powershell
Copy-Item .env.example .env.local
```

В `.env.local` задайте только ключ, без `export`:

```env
OPENROUTER_API_KEY=sk-or-v1-...
```

## Запуск

```powershell
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```powershell
npm run build
npm start
```

## Деплой (Vercel)

В **Settings → Environment Variables** добавьте переменные из `.env.example`, затем выполните **Redeploy**.
